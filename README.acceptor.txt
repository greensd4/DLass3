########################## How To Run #################################

This program gets 2 argument at the command line. The first argument is
a train file that was generated by the gen_examples.py program. The second
argument is the test file (tagged) and we calculate the accuracy on it.
Commnads:
    To create the examples and train and test file:
        python gen_examples.py
        # please notice that the train and test files will be called
        # "pos_neg_test" and "pos_neg_train"

    To run the program:
        python experiment.py <train file> <test file>
